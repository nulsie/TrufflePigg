Uses

This website is written in HTML, CSS and JS. It doesn't use any trackers or analytics systems. It uses a free DuckDNS subdomain and is hosted on GitHub pages but we're looking forward on transitioning to Codeberg pages, the forum uses the free version of Cbox(theme:transparent), and the ideas page uses Giscuss. The badge was created using Datakra.sh's button maker. This website was written in Neovim. The feed system used in the site is a hobbyist Hy-based markup format i'm developing called HYML or HY-based Markup Language.
