TrufflePigg

TrufflePigg is a sophisticated reconnaissance, fuzzing, and OSINT tool built in Node.js for domain and server analysis. This is the project's and tool's page.

Installation

Git Clone:

Codeberg:

git clone git@codeberg.org:nulsie/TrufflePigg.git
cd TrufflePigg

GitHub:

git clone https://github.com/nulsie/trufflepigg.git
cd trufflepigg

npm (GitHub Package):

npm install -g @nulsie/trufflepigg
