Prerequisites & Installation

TrufflePigg relies on Node.js(obviously) for it to work, so see that it is installed.

the tool currently uses 4 dependencies:

  • Cheerio ^1.0.0-rc.12
  • P-limit ^3.1.0
  • Socks ^2.8.3
  • Socks-Proxy-Agent ^8.0.3

Clone the repository:

Codeberg:

git clone git@codeberg.org:nulsie/TrufflePigg.git
cd TrufflePigg

GitHub:

git clone https://github.com/nulsie/trufflepigg.git
cd trufflepigg

or

Through npm(GitHub Package)

npm install -g @nulsie/trufflepigg

For help, just command:

--guide, guide, help or h

To make it work, just command:

trufflepig [configuration commands if you are using them]
